# MobileFaceNet
  论文 ： [MobileFaceNets: Efficient CNNs for Accurate Real-time Face Verification on Mobile Devices](https://arxiv.org/abs/1804.07573) 的AMsoftmaxLoss实现

## 环境需要

- Amsoftmax的配置

- DepthwiseConvolution层的添加

## 训练数据集
  
  VggFace2数据集，图像size：112X96
